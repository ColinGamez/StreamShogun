# Getting Started with StreamShōgun

Welcome to **StreamShōgun** — the modern IPTV player for desktop.

## System Requirements

| Platform | Minimum |
|----------|---------|
| Windows  | Windows 10 64-bit or later |
| macOS    | macOS 12 (Monterey) or later |
| Linux    | Ubuntu 20.04+ / Fedora 36+ (x64) |

StreamShōgun uses **Electron** under the hood, so any modern 64-bit OS should work.

## Installing

1. Download the latest installer from [GitHub Releases](https://github.com/stream-shogun/stream-shogun/releases).
2. Run the installer for your platform (.exe, .dmg, or .AppImage).
3. Launch **StreamShōgun** — you'll see the Welcome screen.

## First-Run Walkthrough

When you open StreamShōgun for the first time:

1. **Welcome Screen** — click **"Add Playlist"** to get started.
2. Navigate to **Library** (📚 in the sidebar) to manage your playlists and EPG sources.
3. Once a playlist is loaded, switch to **Channels** (📡) to browse your channels.
4. Click any channel to start watching in the **Player** (▶️).

## Creating an Account (Optional)

StreamShōgun works fully offline. Creating an account enables:

- **Cloud Sync** (Pro) — sync settings, favourites, and history across devices.
- **Billing management** — upgrade/downgrade your plan.

To create an account:
1. Go to **Settings** → **Account**.
2. Click **"Sign Up"** or **"Log In"**.

## Common Issues

| Problem | Fix |
|---------|-----|
| App won't start | Ensure your OS meets the minimum requirements. Try running as administrator (Windows). |
| White/blank screen | Delete the app cache folder and restart. |
| No channels showing | You need to add at least one M3U playlist — see [Adding Playlists](adding-playlists). |

## When to Contact Support

If the app crashes on start-up repeatedly, or if the installer fails, please contact support with your OS version and the error message.
