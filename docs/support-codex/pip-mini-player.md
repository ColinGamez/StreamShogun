# Picture-in-Picture (PIP) Mini Player

**PIP mode** lets you watch a stream in a small floating window while using other applications. This is a **Pro** feature.

## How to Enable PIP

1. Start playing a channel in the **Player** (▶️).
2. Look for the **PIP button** in the player controls (usually a small overlapping-squares icon).
3. Click it to launch the mini player.

A small floating window will appear with your stream. You can:
- **Move** it anywhere on screen by dragging.
- **Resize** it by dragging the edges.
- **Close** it by clicking the X or pressing the PIP button again.

## Requirements

- **Pro subscription** — PIP is a Pro-only feature. [Learn about upgrading](subscriptions-billing).
- PIP uses a separate Electron window, so it works across all desktops/workspaces.

## Common Issues

| Problem | Fix |
|---------|-----|
| PIP button not visible | Ensure you have an active Pro subscription. |
| PIP window is black | The stream may have ended. Try a different channel. |
| PIP window won't stay on top | Check your OS window manager settings. On some Linux DEs, "always on top" may need explicit permission. |
| Audio plays in both windows | Only the PIP window should play audio. If both play, restart the app. |

## Disabling PIP

To close the PIP window:
- Click the **X** on the PIP window, or
- Click the PIP button again in the main player controls.
